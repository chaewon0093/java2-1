자바의 특성

소스(.java)와 클래스(.class)파일
하나의 소스 파일에 여러 클래스 작성 가능(public 클래스는 하나만 가능)
소스 파일의 이름과 public으로 선언도니 클래스 이름은 같아야 함
클래스 파일에는 하나의 클래스만 존재(다수의 클래스를 가진 자바 소스를 컴파일하면 클래스마다 별도 클래스 파일 생성)
